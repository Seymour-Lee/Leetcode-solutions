## 589. N-ary Tree Preorder Traversal
This problem gives us a data structure named N-ary Tree
```C++
// Definition for a Node.
class Node {
public:
    int val;
    vector<Node*> children;

    Node() {}

    Node(int _val, vector<Node*> _children) {
        val = _val;
        children = _children;
    }
};
```
